📊 Cash Flow Dashboard (Power BI)
📌 Overview

This project presents a financial dashboard to analyze cash flow, including inflow, outflow, and net cash position.

🎯 Objectives
- Monitor business cash flow performance
- Identify key drivers affecting cash balance
- Support financial decision-making
🛠 Tools Used
- Power BI
- SQL
- Excel

📈 Key Features
- Cash Flow Waterfall Analysis
- Inflow vs Outflow Breakdown
- Cash Burn Rate Tracking
- Contribution Margin Analysis
📷 Dashboard Preview

🔍 Key Insights
- Revenue is the main driver of cash inflow
- COGS represents the largest cost component
- Operating expenses significantly impact profitability
